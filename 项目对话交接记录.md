# MineShark 项目对话交接记录

更新时间：2026-05-08

## 1. 项目定位

当前项目定位为：

```text
懂安全场景的 AI 应用开发项目
```

不是单纯的模型训练 demo，也不是凭空包装的大模型 Agent 项目，而是在真实安全监测底座上补充 AI 分析和大模型辅助研判能力。

项目主叙事：

```text
在 Wazuh / Suricata / Zeek 安全监测底座上，补充 AI 流量检测模块与大模型辅助研判能力，形成从日志采集、模型识别到中文审计报告生成的 AI 安全应用原型。
```

## 2. 已确认的真实项目经历

用户此前真实完成过：

- 在虚拟机中安装 Ubuntu 22.04 最小化命令行系统。
- 从 Wazuh 官网一体化部署 Wazuh。
- 部署 Suricata、Zeek 两个流量分析/日志组件。
- 打通 Wazuh、Suricata、Zeek 的安全日志采集、分析和展示链路。
- 在宿主机开发 Python AI 分析模块，为系统补充“模型大脑”。
- AI 模块包括 MineShark/Zeek 风格日志解析、PyTorch Transformer 模型训练、checkpoint 保存和推理。

此前简历中关于 LLM Agent / RAG / 中文报告的描述偏超前，尚未完整实现。因此本轮开发目标是补齐一个可运行的轻量级安全分析报告生成器，而不是虚构完整 SOC Agent 平台。

## 3. 简历目标

项目最终服务于：

- AI 应用开发岗
- Agent 开发岗
- 后端开发岗

但第一标签应当是：

```text
懂安全场景的 AI 应用开发
```

简历推荐表述：

```text
基于 DeepSeek API 构建安全分析报告生成器，将 Transformer 模型检测结果、Zeek/MineShark 流量上下文与本地安全知识库检索结果融合，自动生成包含风险等级、证据摘要、可疑原因与排查建议的中文安全审计报告。
```

边界表述：

```text
轻量级安全分析 Agent / 报告生成器
```

不要包装成：

```text
完整自动化安全运营平台
```

## 4. 本轮补实现的需求边界

第一版只做 CLI 安全分析报告生成器。

不做：

- 自动封禁
- 自动处置
- FastAPI 服务
- Wazuh API 写回
- 多 Agent 平台

要做：

```text
Zeek/MineShark 日志
→ Transformer 模型推理
→ 提取高风险连接上下文
→ 本地安全知识库检索
→ DeepSeek API 或规则兜底报告生成
→ 输出 JSON / Markdown
```

报告对象：

```text
安全运营人员
```

报告重点：

- 风险等级
- 事件摘要
- 可疑依据
- 可能安全含义
- 排查建议
- 误报与局限性提示

- 支持 `--no-llm` 规则兜底模式
- 输出结构化 JSON 和中文 Markdown 报告

## 6. 关键运行命令

无大模型兜底测试：

```powershell
python .\agent_reporter\agent_audit.py --checkpoint checkpoints/main_in_domain.pt --log-file logs_malware/Zeus.pcap.log --max-events 5 --no-llm
```

DeepSeek API 模式：

```powershell
$env:DEEPSEEK_API_KEY="你的key"
$env:DEEPSEEK_BASE_URL="https://api.deepseek.com"
$env:DEEPSEEK_MODEL="deepseek-chat"
python .\agent_reporter\agent_audit.py --checkpoint checkpoints/main_in_domain.pt --log-file logs_malware/Zeus.pcap.log --max-events 5
```

MTA 日志测试示例：

```powershell
python .\agent_reporter\agent_audit.py `
  --checkpoint checkpoints/main_in_domain.pt `
  --log-file data/mta/logs_new_malware/2026-01-31-traffic-analysis-exercise.pcap.log `
  --threshold 0.9 `
  --max-events 5 `
  --output-json agent_reporter/outputs/mta_audit.json `
  --output-md agent_reporter/outputs/mta_audit.md
```
